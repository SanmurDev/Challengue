# Realizar conversiones entre diferentes monedas

Con esta aplicacion podemos realizar las siguientes conversiones:

1 - Colones     ->      Dolares
2 - Dolares     ->      Colones
3 - Colones     ->      Euros
4 - Euros       ->      Colones
5 - Dolares     ->      Euros
6 - Euros       ->      Dolares

La aplicacion se conecta a una API con un codigo y realiza la conversion que necesites
